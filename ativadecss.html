Funções de CSS

min()

O min() no CSS serve para pegar o menor valor entre os valores que você colocou.

É como se o CSS comparasse os números e escolhesse automaticamente o menor deles.

width: min(500px, 90%)
font-size: min(5vw, 40px);

max()

O max() no CSS faz o contrário do min().

Ele serve para pegar o maior valor entre os valores que você colocou.

width: max(300px, 50%);
font-size: max(18px, 2vw);

calc()

O calc() no CSS serve para fazer cálculos diretamente no CSS.

Com ele você consegue:

  • somar (+)
  • subtrair (-)
  • multiplicar (*)
  • dividir (/)
width: calc(100% - 200px);
height: calc(100vh - 80px);

clamp()

O clamp() no CSS serve para definir:

  • um valor mínimo
  • um valor ideal
  • um valor máximo

Ele é muito usado em responsividade, principalmente para fontes e tamanhos.

font-size: clamp(16px, 5vw, 40px);

Isso significa:

  • nunca menor que 16px
  • tenta usar 5vw
  • nunca maior que 40px

Largura:

width: clamp(300px, 80%, 1200px);
